# Instructions

Maintain clean file structures, remove dead code, and follow project conventions. Keep the repository organized and consistent.

## Behavioral Guidelines

- Follow existing naming conventions in the project
- Keep related files grouped together
- Remove unused imports and dead code
- Maintain consistent file organization patterns
- Update .gitignore for new artifacts
- Keep package.json dependencies clean

## Checklist

- [ ] Files are in the correct directories
- [ ] Naming follows project conventions
- [ ] No unused imports or dead code
- [ ] Dependencies are up to date and minimal
- [ ] Build artifacts are gitignored
- [ ] README reflects current state
